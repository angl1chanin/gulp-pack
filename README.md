# Gulp сборка для верстки
## Описание плагинов
```
HTML:
    - file-include     - подключение дополнительных html файлов к основному
    - htmlmin          - сжатие HTML
    - webphtml         - добавляет тег для webp изображения поверх тега с другим расширением (jpg, png, jpeg)
    - size             - выводит размер до/после сжатия

SCSS:
    - sass             - препроцессор
    - autoprefixer     - добавляет дополнительные префиксы для разных браузеров к css свойствам
    - stripcsscomments - удаляет комментарии из css файлов
    - csso             - сжатие стилей
    - rename           - переименовывание
    - webpcss          - добавляет свойства для подключение изображений webp, если браузер поддерживает

JavaScript:
    - terser           - сжатие JS
    - size             - выводит размер до/после сжатия

IMG:
    - imagemin - сжатие изображения
    - newer - отслеживает появление новых файлов, чтобы не сжимать ранее сжатые снова
    - webp - конвертирует изображение дополнительно в формат webp
```

## Структура папок и файлов
```
├── src/                          # Исходники
│   ├── js                        # Скрипты
│   │   └── main.js               # Главный скрипт
│   ├── scss                      # Стили сайта (препроцессор sass в scss-синтаксисе)
│   │   └── main.scss             # Главный файл стилей
│   │   └── _base.scss            # Файл для базовой настройки сайта
│   │   └── _fonts.scss           # Файл для подключения шрифтов (можно использовать миксин)
│   │   └── _mixins.scss          # Файл для подключения миксинов из папки mixins
│   │   └── _vars.scss            # Файл для написания css- или scss-переменных
│   │   └── _reset.scss           # Файл для cброса стилей
│   │   └── _index.scss           # Файл для стилей главной страницы
│   │   ├── components            # scss-компоненты
│   ├── img                       # папка для хранения изображений
│   └── html                      # HTML файлы
│   │   └── chunks                # Подключаемые HTML файлы
│   │   └── index.html            # Файл разметки главной страницы
└── gulpfile.js                   # файл с настройками Gulp
└── package.json                  # файл с настройками сборки и установленными пакетами
└── README.md                     # документация сборки
```

## Команды для запуска/работы
Команды прописаны в файле ***package.json***
```
npm start     - запускает сервер, выполняет все задачи
npm run build - выполняет сборку проекта
```
**Sourcemaps** (для js, scss) включены **только** во время разработки

### Директории
Изменить можно в ***gulpfile.js/gulp/config/path.js***
```
src/  - директория с основными файлами проекта
dist/ - директория сборки файлов в процессе работы, а также окончательной сборки
```
